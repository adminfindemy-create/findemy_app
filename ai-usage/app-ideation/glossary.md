# Glossary

Shared vocabulary for talking about Findemy. Use these terms consistently across product, design, and engineering.

| Term | Meaning |
|------|---------|
| **Academy** | A provider of skill classes — an independent guru, a studio, or an institute. The core supply unit on Findemy. |
| **Coach / Mentor** | An instructor at an academy who teaches one or more batches. |
| **Category / Skill** | The discipline being taught — e.g. music, dance, art, yoga, fitness. Used for browsing and filtering. |
| **Batch** | A recurring class group at an academy (a specific course + level + schedule). What a learner ultimately **enrols** in. |
| **Slot** | A specific bookable time for a **trial** (date + time + capacity). |
| **Trial** | A low-commitment demo class a learner books to experience an academy before enrolling. The core conversion point. |
| **OTP (at class)** | The code a learner shows at the academy so attendance for a trial/class can be verified. |
| **Enrollment** | A learner joining a recurring **batch** after deciding to commit. Can be renewed or paused. |
| **Workshop** | A one-off or short-term session hosted by an academy — masterclass, demo, or special class — that learners can register for. |
| **Event** | A happening surfaced in the app that learners can register for. (Findemy-run marquee competitions are a *future* expansion — see scope note in `05-mvp-scope.md`.) |
| **Review** | A learner's rating/feedback on a trial or academy; builds trust and reputation. |
| **Student app** | The learner-facing mobile app (discover, book, enrol, manage). |
| **Admin / Academy app** | The academy-facing mobile app (profile, batches, trials, attendance, enrolments). |
| **Commission / Platform fee** | Findemy's cut on a trial booking or enrolment — the MVP revenue model. |
| **Delhi-NCR** | The launch market: Delhi and the National Capital Region. The only market in the MVP. |
| **Reels** | *(Future)* Short-form learner content; part of the creator ecosystem, not in the MVP. |
| **Leaderboard** | *(Future)* Gamified rankings (academy/city/national); not in the MVP. |
