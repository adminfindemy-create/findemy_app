# S0.1 — Handoff: shared contract & conventions

> Backend → both apps. The shared vocabulary every later slice imports from **`@findemy/types`** (`backend/shared/types/src/index.ts`).

## Exported enums (new in S0.1)
| Export | Values | Use |
|---|---|---|
| `ModeEnum` / `Mode` | `'in-studio' \| 'online'` | batch/trial **delivery mode** (wire). DB `BatchMode` stores `online\|offline` (`offline`≡in-studio); API maps on read. **Option A — no DB migration.** |
| `PlanEnum` / `Plan` | `'monthly' \| 'quarterly' \| 'annual'` | subscription term (`biannual` removed in S0.2) |
| `AttendanceModeEnum` / `AttendanceMode` | `'otp' \| 'qr' \| 'auto_join'` | trial→otp, in-studio→qr, online→auto_join |
| `ApiErrorCodeEnum` / `ApiErrorCode` | 20 codes (VALIDATION, UNAUTHORIZED, …) | switch on `error.code` |

## Conventions (rails)
- **Money** = integer **paise** (`_paise`/`Paise` suffix); server-authoritative — clients never compute/send amounts.
- **Discount** = **bps** (`Bps` suffix); product cap **3000 (30%)**.
- **Time** = store/transport UTC; render IST client-side.
- **Error envelope** = `{ error: { code, message, fields? } }` (already enforced by `backend/api` `plugins/error-handler.ts` + `lib/app-error.ts`).
- **Login/auth OTP** = 6-digit (`OtpVerifyRequest.code` length 6) — **separate** from the **attendance OTP** which becomes 4-digit in S2.2.
- **Typed client**: apps call through `@findemy/api-client` `createClient` (`student-app/src/lib/api.ts`, `academy-app/src/lib/api.ts`).

## Apps: how to consume
```ts
import { Mode, Plan, AttendanceMode, ApiErrorCode, ModeEnum } from '@findemy/types';
```
Use these instead of inline string unions. **Don't** re-declare a shared shape in an app.

## Known follow-ups (not S0.1)
- `Batch.mode` is still serialized raw (`online|offline`) — **S1.3** normalizes the batch read serializer to emit canonical `Mode` (`'in-studio'|'online'`); `Trial.mode` already uses `Mode`.
- App enum-literal adoption happens naturally in the slices that touch each area (S1.3, S2.2, S2.3, …) — S0.1 only publishes the vocabulary; the baseline stays green.
